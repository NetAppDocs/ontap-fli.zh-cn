= 目标受众
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用此内容帮助您将数据从外部阵列迁移到ONTAP 、或者将LUN从NetApp 7-模式阵列过渡到ONTAP。

您应熟悉常见的SAN概念和过程、包括分区、LUN屏蔽、需要迁移其LUN的主机操作系统、ONTAP 以及源第三方阵列。
