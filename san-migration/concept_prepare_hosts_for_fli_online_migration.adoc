= 为 FLI 联机迁移准备主机
:icons: font
:imagesdir: ../media/


[role="lead"]
FLI 联机执行阶段包括准备迁移主机，以便采用受支持的配置。

在许多情况下，可能会在执行此步骤之前执行此修复。如果不是，则可以在此处执行任何主机修复，例如安装主机连接套件或 DSM 。在分析阶段，您将获得一个间隔列表，其中列出了需要在每个主机上执行的项，以便该主机能够使用 ONTAP 实现受支持的配置。根据所执行的迁移类型，可以先修复主机，然后重新启动，也可以仅修复主机。
