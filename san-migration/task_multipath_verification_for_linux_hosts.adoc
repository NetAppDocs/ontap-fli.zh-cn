= Linux 主机的多路径验证
:icons: font
:imagesdir: ../media/


[role="lead"]
在外部 LUN 导入（ Foreign LUN Import ， FLI ）过程中，您应验证主机上是否已配置多路径并正常运行。

对于 Linux 主机，请完成以下步骤。

.步骤
. 要验证在 Linux 主机上是否已配置 DM-MP 多路径并正常运行，请运行以下命令：``multipath -ll``
