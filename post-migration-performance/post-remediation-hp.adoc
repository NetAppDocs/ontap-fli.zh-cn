== 迁移后 - 在线 FLI

删除 LUN 导入后，请检查 LUN 是否设置为 `-space-reserve disabled`和 `-space-allocation enabled`。如果没有，请相应地进行修改。

要使 SCSI UNMAP 功能正常运行，请参阅供应商文档以验证支持的软件版本。需要 Veritas VxVM + VxFS 组合来启用此功能。

[cols="4"]
|===
| Filesystem | 块设备 | LVM（文件系统） | Sanboot 


| 步骤 1：重新扫描所有映射的 HP-UX 主机上的存储 | 步骤 1：重新扫描所有映射的 HP-UX 主机上的存储。 | 步骤 1：重新扫描所有映射的 HP-UX 主机上的存储 | 步骤 1：重新扫描所有映射的 HP-UX 主机上的存储 
|===
.步骤
. 识别存储设备。
+
[source, cli]
----
ioscan -fnC disk
insf -e
sanlun lun show all
sanlun lun show all -pv
sanlun lun show -wwpn
----
. 导入 VG 并使 VG 联机。
+
[source, cli]
----
vgimport /dev/vg_name /dev/disk/disk_name
vgchange -a y /dev/vg_name
----
. 标识卷组和文件系统。
+
[source, cli]
----
vgdisplay -v /dev/vg_name
vgdisplay /dev/vg_name
lvdisplay /dev/vg_name/lv_name
----
. 挂载文件系统。
+
[source, cli]
----
bdf
mount /dev/vg_name/vol_name /mount_point
----
