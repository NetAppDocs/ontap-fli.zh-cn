= 执行 ONTAP FLI Solaris 迁移后主机修复
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
迁移后，Solaris 主机需要采取补救措施，以确保新导入的 LUN 被正确识别并正常工作。



== 迁移后 - 离线 FLI

对于离线 FLI，删除 LUN 导入并确认主机启动器已连接后，使 LUN 联机。

NOTE: 有关最新和更新的命令列表，请参阅操作系统供应商文档。

.步骤
. 连接后重新扫描。
+
[source, cli]
----
cfgadm -al
----
. 识别存储设备。
+
[source, cli]
----
format
sanlun lun show all
mpathadm list lu
mpathadm show lu
----
. 验证设备是否已连接。
+
[source, cli]
----
format |grep disk_id
----
. 导入 zpool，FS 将自动挂载。
+
[source, cli]
----
zpool import zpool_name
df -g (to ensure its unmounted)
----
